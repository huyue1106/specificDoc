QQ炫舞&炫舞梦工厂重构规范

此规范只适用于QQ炫舞及炫舞梦工厂的重构规范，如与CP规范冲突，以此规范为主。

移动端分享代码

如页面需放到小灵通APP的，需在页面上添加分享按钮并添加以下分享js代码：

function share() {
    if (typeof(GameHelper) != "undefined" && typeof(GameHelper.shareWebPageWithFuntion) != "undefined") {
        vartitle = '炫舞小灵通”新人礼包！', //分享标题
        desc = '永久套装，非卖道具专属新人厚礼！', //分享文案
        icon = 'https://ossweb-img.qq.com/images/bangbang/act/a201600812x5/200.png', //分享图片url
        url = 'https://x5.qq.com/cp/a20171013xlt/index.htm'; //分享链接，如QQ炫舞:https://x5.qq.com/cp/专题活动名称(文件夹名称)/文件名
        GameHelper.shareWebPageWithFuntion(title, desc, icon, url, '1,2,3,4,5', 1);
    } else {
        alert("请在助手APP参与本次活动");
    }
}

除了放小灵通的页面，其余页面都需要添加常规分享组件

另外页面内有登录组件的话，需在页面<head></head>中引入<script src="https://open.mobile.qq.com/sdk/qqapi.https.js?_bid=2389"></script>

分离地址

QQ炫舞：//game.gtimg.cn/images/x5/cp/专题活动名称(文件夹名称)/图片名称.jpg

QQ炫舞梦工厂：//game.gtimg.cn/images/mgc/cp/专题活动名称(文件夹名称)/图片名称.jpg

图片标签

图片标签必须写上宽度、高度和alt属性，宽高为图片的原大小

<img src="" width="500" height="200" alt="活动奖品建筑排程卡"/>
