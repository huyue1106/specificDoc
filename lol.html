K6重构规范

此规范只适用于K6英雄联盟项目的重构规范，如与CP规范冲突，以此规范为主

整体通用规范

  • PC端专题通用规范：PC端专题
  • M端专题通用规范：移动端专题

分离地址

-项目文件夹名称命名规范：禁止全拼或者拼音字母，一律小写，一律使用时间英文名。

规范：a+时间+英文名

-页面https升级，所有分离路径(js/css/img)都使用//，去掉协议头。

LOL相关专题

PC端分离规则：//game.gtimg.cn/images/lol/act/a20180821seven(专题活动名称/文件夹名称)/图片名称.jpg

M端分离规则：//game.gtimg.cn/images/lol/m/act/a20180821seven(专题活动名称/文件夹名称)/图片名称.jpg

PC端分离规则：//game.gtimg.cn/images/lpl/es/act/a20170712canyon/(专题活动名称/文件夹名称)/图片名称.jpg

M端分离规则：//game.gtimg.cn/images/lpl/es/m/act/a20170712canyon(专题活动名称/文件夹名称)/图片名称.jpg

function is_pc(){
    var os = new Array("Android","iPhone","Windows Phone","iPod","BlackBerry","MeeGo","SymbianOS");  // 其他类型的移动操作系统类型，自行添加
    var info = navigator.userAgent;
    var len = os.length;
    for (var i = 0; i < len; i++) {
        if (info.indexOf(os[i]) > 0){
            return false;
        }
    }
    return true;
}
